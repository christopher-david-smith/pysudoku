# SudokuSolver
Sudoku solver using Python3 and bruteforce

### Usage
```py
>>> from sudokusolver import SudokuSolver
>>> solver = SudokuSolver([
...     [2, 0, 0, 8, 0, 4, 0, 0, 6],
...     [0, 0, 6, 0, 0, 0, 5, 0, 0],
...     [0, 7, 4, 0, 0, 0, 9, 2, 0],
...     [3, 0, 0, 0, 4, 0, 0, 0, 7],
...     [0, 0, 0, 3, 0, 5, 0, 0, 0],
...     [4, 0, 0, 0, 6, 0, 0, 0, 9],
...     [0, 1, 9, 0, 0, 0, 7, 4, 0],
...     [0, 0, 8, 0, 0, 0, 2, 0, 0],
...     [5, 0, 0, 6, 0, 8, 0, 0, 1]
...])
>>> solver.solve()
>>> solver.pretty_print()
```
